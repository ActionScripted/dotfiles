Vimwiki Cheatsheet
===

Movement
---

| Command     | What it do                           |
|-------------|--------------------------------------|
| <BACKSPACE> | go back to index                     |
| <ENTER>     | create/follow a link                 |
| <TAB>       | in visual mode, move between links   |
| =           | make current line a heading          |
| \ww         | start Vimwiki (how'd you get here?!) |


Tables
---
For a 3x3 table:
    VimwikiTable 3 3

In insert mode, you can <TAB> between table cells
